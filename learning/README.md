# Solidity Learning Path (Avalanche C-Chain)

Numbered lessons to learn Solidity basics. Each folder has a contract you can read, compile, and deploy.

| # | Topic | Contract |
|---|--------|----------|
| 1 | Intro | Minimal first contract |
| 2 | State Variables | Persistent on-chain storage |
| 3 | Local Variables | Function-scoped variables |
| 4 | Function | view, pure, state-changing |
| 5 | Constructor | One-time deployment logic |
| 6 | Data Types | uint, bool, address, string, bytes |
| 7 | Array | Fixed and dynamic arrays |
| 8 | Loops | for loops (gas-aware) |
| 9 | Conditionals | if/else, require, ternary |
| 10 | Struct | Custom data structures |
| 11 | Mapping | Key-value storage |
| 12 | Storage Locations | storage, memory, calldata |
| 13 | Global Variables | msg, block, tx |
| 14 | Contract Balance | Receiving AVAX, balance |
| 15 | Visibility | public, private, internal, external |
| 16 | Exercises | Combine concepts (owner + cap) |

**Compile everything:** `npm run compile`  
**Deploy (e.g. Greeter):** `npm run deploy:fuji`
